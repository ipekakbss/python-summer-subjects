# Python Summer Subjects 

Hi! This is a personal space where I keep my Python practices and mini-scripts. I'm using this repository to track my progress and improve my coding skills over the summer.

## What's in here?

* **`warm_up.py`**: Just some basic syntax practice to get started.
* **`speed_stats.py`**: A simple script to calculate mean, min, max, and standard deviation using the built-in `statistics` module (takes 10 fixed inputs).
* **`speed_stats_v2.py`**: The improved version! I added dynamic user input, automatic file saving to `speeds.txt`, and wrote my own standard deviation function from scratch instead of relying on the built-in module.

## What I'm currently practicing
Throughout these practices, I am focusing on:
- User inputs and type casting (`input`, `int`, `float`)
- Loops (`for`, `while`)
- List operations (`append`)
- File I/O operations (`with open(...) as file`, `write`)
- Defining custom functions (`def`) and implementing mathematical formulas.
